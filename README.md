# 🪙 InfoCrypto-App

![Android](https://img.shields.io/badge/Platform-Android-brightgreen.svg) 
![Kotlin](https://img.shields.io/badge/Language-Kotlin-blue.svg)
![API](https://img.shields.io/badge/Data-Real--Time%20API-orange.svg)

**InfoCrypto-App** es una aplicación nativa de Android diseñada para el monitoreo dinámico del mercado de criptomonedas. Los usuarios pueden visualizar cientos de activos digitales, sus precios actuales y el comportamiento del mercado en tiempo real.

---

## Description / Descripción

**English:**
> **CryptoTracker:** A native Android app powered by a real-time API to monitor hundreds of cryptocurrencies. Features live pricing and 24-hour price change analysis (gains/losses).

**Español:**
> Una aplicación Android potente que consume datos de una API externa para mostrar el estado actual de las criptomonedas más importantes del mundo, permitiendo ver de un vistazo si el mercado está al alza o a la baja.

---

## Características Principales

-   **Listado Completo:** Acceso a una vasta base de datos de criptomonedas.
-   **Precios en Vivo:** Actualizaciones constantes mediante integración con API REST.
-   **Análisis de 24 Horas:** Indicadores visuales claros sobre la subida o bajada porcentual en el último día.
-   **Interfaz Moderna:** Diseño limpio, intuitivo y adaptado a las pautas de Material Design.
-   **Búsqueda Rápida:** Encuentra tu moneda favorita en segundos.

---

## Stack Tecnológico

Este proyecto utiliza las mejores prácticas y herramientas modernas para el desarrollo Android:

-   **Lenguaje:** [Kotlin](https://kotlinlang.org/) (o Java).
-   **Arquitectura:** MVVM (Model-View-ViewModel) para una separación de conceptos clara.
-   **Networking:** [Retrofit](https://square.github.io/retrofit/) + OkHttp para el consumo de la API.
-   **Gestión de Datos:** Coroutines para tareas asíncronas y flujo de datos fluido.
-   **Interfaz:** XML / Jetpack Compose (según tu implementación).
-   **Imágenes:** [Glide](https://github.com/bumptech/glide) o [Coil] para cargar los logos de las criptomonedas.
